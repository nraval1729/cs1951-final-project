Writeup: Lab 4

  • You have n tweets that you'd like to classify. m of the n tweets are positive tweet, while the remaining are negative. You have a classifier that assigns any tweets to be positive. Given this scenario, what are the precision and recall values (in terms of m and n) for your classifier?
    • Precision = TP / (TP + FP) = m / (m + (n - m)) = m / n
    • Recall = TP / (TP + FN) = m / (m + 0) = m / m = 1
  • Now, consider the case where you always assign each tweet to be negative. What will the precision and recall values be?
    • Precision = TP / (TP + FP) = 0 / (0 + 0) = undefined
    • Recall = TP / (TP + FN) = 0 / (0 + m) = 0
